🖥️ PC Showcase - 本地化快速部署电脑展示系统
一款轻量级、零依赖的电脑配置展示平台，适用于装机店、硬件测评、线下展会等场景。纯 PHP + MySQL + HTML5 构建，无需框架，一键启动。

✨ 功能特性
电脑配置分类展示（显卡、显示器、硬盘、内存、外设）
全局搜索框，支持模糊查询
产品详情页：左侧外观大图，右侧详细参数表
后台管理：分类添加、电脑配置上传（含图片）
响应式布局，PC/平板自适应
极简部署，无编译、无 Composer/NPM
🚀 快速开始
环境要求
PHP 7.4+
MySQL 5.7+（或 MariaDB 10.2+）
任意 Web 服务器（Apache/Nginx/PHP 内置服务器）
部署步骤
克隆项目
git clone https://github.com/你的用户名/pc-showcase.git
cd pc-showcase
